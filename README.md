# FedMDX-for-Medical-Image-Classification


This proposed work introduces FedMDX — a Federated Multi-Modal Deep Learning framework with Explainable AI (XAI) — designed for privacy-preserving medical image classification across heterogeneous institutions. Building upon the limitations identified in the base paper (FedVGM), FedMDX advances the state-of-the-art through five key novelties: a hybrid CNN-ViT encoder with modality-specific adapters, a novel gradient-similarity-filtered aggregation strategy, differential privacy guarantees, a multi-method XAI fusion module, and cross-modality knowledge transfer.
